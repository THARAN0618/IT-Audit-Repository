# IT Infrastructure Audit Dashboard

 Project Overview

This project presents a comprehensive IT audit conducted for a distributor organization. It includes data collection, risk analysis, and a dynamic dashboard to visualize audit results.

 Objectives

* Evaluate IT security and infrastructure
* Identify risks in billing, inventory, and access control
* Provide insights through a visual dashboard

Technologies Used

* HTML, CSS, JavaScript
* Chart.js
* Google Forms (Data Collection)

Features

* Interactive dashboard with charts
* Risk analysis and scoring system
* Section-wise audit insights
* Responsive design

 Dataset

Collected via Google Forms and exported as CSV/JSON.

 Key Insights

* Identified gaps in access control and backup systems
* Highlighted risks in billing and inventory processes
* Provided actionable recommendations
